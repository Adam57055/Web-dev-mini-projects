# Github Profile Finder

## Description
GitHub Profile Finder will fetch the github details like:
- Name of the user
- Number of Repositories
- Bio of the user
- Number of Followers
- Number of Followings
- Profile pic

All just by entering the username and clicking on "Search User".

By clicking on the username, it will redirect to the user's GitHub profile.

## Stacks Used
* JavaScript Promises
* API  

## API Used
https://api.github.com/user/user_name
